# Airbnb_Project

A full-stack web application that visualises Airbnb listing data in New Orleans. 

# Features

- Data visualisation
- Identification of superhosts
- Explore listing details

# Tech Stack

**Backend**\
Node.js | Express

**Database**\
MySQL

**Frontend**\
EJS | HTML | CSS

# Setup Instructions

**1. Clone repository**

git clone https://github.com/val07tine/Airbnb_Project.git
cd Airbnb_Project

**2. Install dependencies**

npm install

**3. Set up environment variables**

Create a .env file in root directory

DB_HOST=localhost\
DB_USER=root\
DB_PASSWORD=password\
DB_NAME=name\
PORT=3000

**4. Import the database**

Create database and import [dump.sql](https://github.com/val07tin/Airbnb_Project/blob/main/Dump.sql)

**5. Run the application**

npm start
